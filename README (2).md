# Microsoft Copilot Clone

A beautiful and functional web-based AI assistant inspired by Microsoft Copilot.

## Features

- Clean, modern dark UI similar to official Copilot
- Real-time chat interface
- Chat history (Recent chats)
- Smart suggestion prompts
- Responsive design (works on mobile & desktop)
- Fully functional with simulated AI responses

## Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript

## How to Run

1. Clone or download the repository
2. Open `index.html` in any modern browser
3. Start chatting with Copilot!

## Future Enhancements

- Connect to real AI API (OpenAI, Groq, Hugging Face, etc.)
- Save chats to localStorage
- Voice input support
- File upload capability
